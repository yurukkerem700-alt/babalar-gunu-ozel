# 🎉 Babalar Günü Özel

Babanıza özel, duygusal ve unutulmaz bir web sayfası oluşturun!

![Babalar Günü](public/hero-father.jpg)

## ✨ Özellikler

### 📝 7 Adımlı Kolay Form
1. **Babanı Tanıtalım** - Ad, doğum tarihi, meslek
2. **Sevdiği Şeyler** - Hobiler, ilgi alanları, favori renk/yemek/müzik
3. **Kişiliği** - Babanın özellikleri + en güzel anı
4. **Fotoğraflar** - Baba fotoğrafı + birlikte fotoğraf
5. **Sen ve Baban** - İsim, ilişki, hikaye
6. **Mesajın** - Babana özel duygusal mesaj
7. **Tema Seçimi** - 5 farklı tema (Klasik, Sıcak, Doğa, Modern, Deniz)

### 🎨 Babaya Özel Sayfa
- Fotoğraf çerçevesi ile hero section
- "X gündür dünyamızı aydınlatıyorsun" gün sayacı
- Burç, yaş, meslek bilgileri
- Kişilik özellikleri kartları
- Hobiler ve ilgi alanları (emoji ikonlarla)
- Fotoğraf galerisi (altın çerçeveli)
- En güzel anı bölümü
- Dekoratif köşeli büyük mesaj bölümü

### 💝 Paylaşım
- Link kopyalama
- WhatsApp'ta paylaşma
- Sosyal medya paylaşımı

## 🚀 Teknolojiler

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Database**: Supabase (PostgreSQL)
- **Storage**: Supabase Storage
- **Deployment**: Vercel
- **Fonts**: Playfair Display, Nunito, Dancing Script

## 📦 Kurulum

```bash
# Repoyu klonla
git clone https://github.com/yurukkerem700-alt/babalar-gunu-ozel.git
cd babalar-gunu-ozel

# Bağımlılıkları yükle
npm install

# Environment variables ayarla
cp .env.example .env
# .env dosyasını düzenle

# Development server'ı başlat
npm run dev

# Production build
npm run build
```

## 🔧 Environment Variables

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```
